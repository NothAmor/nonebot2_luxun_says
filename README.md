# nonebot2_luxun_says
基于nonebot2机器人框架的鲁迅说插件

# 使用方法
将luxun_says文件夹放置到nonebot2的plugins文件夹中, 并在luxun_says中创建一个名为`says_img`的文件夹, 用以存储生成的图片, 并修改__init__.py文件中第`36`, `46`, `63`行中的路径为你机器中的绝对路径
